mtext.side.h
============

Value
-----

an integer suitable for the side argument of mtext.

Argument
--------

Any value.
`mtext.side.h` is designed to handle arguments in
`c("bottom", "left", "top", or "right")`.

Example
-------

    mtext.side.h("bottom")
    # [1] 1
    mtext.side.h()
    # [1] 3
